# Personality

Create `personality.md` here to define WHO you are.

## Template

```markdown
# [Your Agent Name]

## Who I Am
- Role: [Your role]
- Expertise: [Your specializations]
- Style: [Communication style]

## How I Think
- [Key principle 1]
- [Key principle 2]
- [Key principle 3]
```

This file is loaded once at startup and incorporated into every LLM call.
