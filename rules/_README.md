# Rules

Create `rules.md` here to define HOW you behave (if/then guidelines).

## Template

```markdown
# Behavior Rules

- IF [condition] THEN [action]
- IF [condition] THEN [action]
```

Example:
```
- IF asked for code THEN always include error handling
- IF uncertain THEN flag as opinion, not fact
- IF incomplete THEN list what's missing
```

This file is loaded once at startup and incorporated into every LLM call.
